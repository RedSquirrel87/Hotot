      API Settings

      • Explanation: the parameters above correspond to the number of tweets that will be returned by every request to the Twitter's API. High values can cause low performance in the program (it has to analyze a bigger number of tweets to display them in the right column and position) but they reduce the risks to exceed the limitations imposed by Twitter's API (having more tweet returned for any request there will be less need to do other requests) and the risk of receiving only a portion of new tweets to display. Conversely low values bring to a faster and quick program (it will analyze fewer tweets every time) but carry a high risk of exceeding the time limits imposed by Twitter's API (you need more requests to get a large number of results) and may lead to the acquisition of only a portion of the actual new tweets to display (in case these were more than the set value). So with these two options you can now try different combinations and find the best optimization for your taste.
        Anyway if you are not sure what you're doing, you should leave the default values.
      • Warning: you can not login to your Twitter account with a wrong consumer key/secret so please be careful to change them only if you really know what you're doing. Note also that enabling/disabling this option will reset your access token and so you'll have to get a new one at next login.
      • Keeping enabled this function will contribute indirectly to the development of Hotot+: simply this function automatically adds Hotot+ referral tag to every Amazon url you send or read, that's all. This means nothing changes for the user, since the Amazon prices will remain the same (they will absolutely not grow!) but Amazon will give a small percentage for every sale that occurred from those referral links to Hotot+.This means that you can contribute for free to Hotot+ as if you were making a donation, except that you will not have to pay a penny (in fact, Amazon will pay, not you) and this without having to undergo invasive advertising, potential dangers for your privacy or the inconvenience of waiting or having to click banners. So this is the best way for you to contribute to the development of Hotot+ without even realizing it.
